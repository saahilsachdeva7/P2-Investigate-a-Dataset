The Survived column is the target variable. If Suvival = 1 the passenger survived, otherwise he's dead. The other variables that describe the passengers are: PassengerId: and id given to each traveler on the boat Pclass: the passenger class. It has three possible values: 1,2,3 The Name The Sex The Age SibSp: number of siblings and spouses traveling with the passenger Parch: number of parents and children traveling with the passenger The ticket number The ticket Fare The cabin number The embarkation. It has three possible values S,C,Q

The count variable shows that 177 values are missing in the Age column. One solution is to replace the null values with the median age which is more robust to outliers than the mean

Conclusions Females and children were rescued first. Passengers with expensive tickets were saved first. people with age between 15-50 and having low cost tickets were among the people who died most.

Limitations of the study There are the variables that I don't analyse (SibSp, Parch, cabin, etc.). These variables have bias (based on the description of the data; ex.: With respect to the family relation variables (i.e. sibsp and parch) some relations were ignored.) or significant amount of data is missed (like for Cabin). To avoid adding noice to the analysis, I've decided to exclude them from the analysis. Handling missing values. The dataset is filled with missing values of the age. The missing age values are imputed with the median and here is the comparison between default (dropped missing values) and imputed age values

Using sample instead of the population. It's known, that there were 1317 passengers and 885 crew members on the board of Titanik. Our data set represent just 891 persons. There is no clear mention of the crew members in the data set - so either they are not in the data set or they are hidden among the passengers of the 1, 2 and 3 classes. If the crew memebers are represented in our data set, the overal statistic might be distorted.

we cannot be sure of these results without hypothesis testing. we can do t-test to be sure of these results. This dataset was at least in part collected after the titanic sank. This means that there's significant survivor bias.
